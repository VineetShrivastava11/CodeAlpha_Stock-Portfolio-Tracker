# 📈 Stock Portfolio Tracker (Python)

A simple Python-based Stock Portfolio Tracker that calculates the total investment value based on user-entered stock names and quantities. The program uses a predefined dictionary of stock prices and performs basic calculations through a console interface.

## 📌 Features

- User inputs stock names and quantities.
- Uses a predefined dictionary to store stock prices.
- Calculates the total investment value.
- Displays the investment summary.
- Optionally saves the results to a .txt or .csv file.

## 🛠️ Technologies Used

- Python
- Dictionary
- User Input/Output
- Basic Arithmetic
- File Handling (Optional)
   
## 📂 Project Structure


stock-portfolio-tracker/
│── stock_portfolio_tracker.py
│── README.md

│── portfolio.csv (Optional)


## 🎯 How It Works

- The program stores stock prices in a predefined dictionary.
- The user enters the stock name and quantity.
- The program calculates the investment value for each stock.
- Finally, it displays the total investment value.
